# Network Engineering & Security Labs

This repository contains hands-on network engineering and network security labs designed to demonstrate practical skills in designing, configuring, and troubleshooting network infrastructures.

The labs are implemented using Cisco Packet Tracer and related simulation tools, covering key concepts such as switching, routing, redundancy, and security.

---

## 🧪 Labs Included

- Spanning Tree Protocol (STP) – Loop prevention and redundancy
- VLAN Configuration – Network segmentation and isolation

-------------

## 🛠️ Tools & Technologies

- Cisco Packet Tracer  
- Cisco IOS (simulated)  
- Networking protocols (STP, VLAN)
- Network troubleshooting techniques  

---

## 🎯 Objectives
